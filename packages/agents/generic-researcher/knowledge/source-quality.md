# Source quality ranking

Rank sources in roughly this order:

1. **Primary** — the thing itself. Official filings (SEC, KIND), academic
   papers, original datasets, press releases from the subject.
2. **High-trust secondary** — Reuters / Bloomberg / FT / WSJ / Nikkei,
   peer-reviewed reviews, trade publications with byline + dateline.
3. **Analyst reports** — cite the house (Gartner, IDC, McKinsey). Note that
   figures may be estimates; mark as such.
4. **Wikipedia** — useful for dates and high-level facts; NOT a primary
   source. Always drill through to its citations.
5. **Blogs / social** — weakest. Only use when the author is the subject
   (CEO's own blog, maintainer's repo README).

## Flags to note

- **Recency**: anything more than 12 months old for fast-moving topics
  (AI, crypto, geopolitics) should be called out.
- **Conflict of interest**: vendor-sponsored research, consultancy's own
  customers — note it.
- **Translation**: non-English sources, flag and quote verbatim plus
  translation when possible.
