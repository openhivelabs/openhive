# House style

Defaults — override per-brief when the audience or publication requires it.

## Voice

- Active over passive unless the agent is unknown ("mistakes were made" only
  when obscuring who made them is the point).
- Short sentences by default. Vary length for rhythm.
- No corporate filler: avoid "leverage", "synergy", "best-in-class".

## Numbers

- Spell out one through nine in prose; use digits for 10+.
- Always include the unit ("$2.3M", "45%", "3 seconds").
- Relative change: "+18% QoQ", not "a gain of 18 percent quarter-over-quarter".
- Dates: ISO when precision matters (2026-04-20), otherwise "April 20, 2026".

## Formatting

- Headings: Title Case for document titles, Sentence case for H2 and below.
- Lists: bullets for items; numbered only when order matters.
- Bold for emphasis sparingly — if everything is bold, nothing is.
- Code / identifiers / filenames in `monospace`.

## Transitions

- Avoid "moreover", "furthermore", "additionally" — cut them and the prose
  usually flows fine.
- Use "but" and "and" at sentence starts when natural.
