## Format by source type

A summary's shape should match the source. The wrong shape loses information even when the words are right.

### Meeting / call

Return three sections, in order:

1. **Decisions** — what was decided. Each as a sentence with the decision-maker named when it matters.
2. **Action items** — `Owner — action — by date`. One per line. If owner or date is missing, mark `(owner TBD)` or `(no date)`; do not invent.
3. **Open questions** — what was raised but not resolved. Useful for the next meeting's agenda.

Do not start with "the meeting was about…". Lead with decisions.

### Long document (report, paper, memo)

Two-tier structure:

- **TL;DR** — one paragraph, 3–5 sentences, that delivers the central claim and the strongest support.
- **Section bullets** — one bullet per major section, each capturing the section's load-bearing point. Optional — only if asked for more detail than the TL;DR.

Preserve the document's structure where it carries meaning; flatten it where it doesn't.

### Email or chat thread

Format:

- **Status** — one sentence: where the thread currently stands.
- **Positions** — who said what, named, in chronological order if it matters; clustered by stance if it doesn't.
- **Outstanding** — what's awaiting whom.

Distinguish "agreed" from "no objection raised yet" — they look the same in writing but mean different things.

### Article or blog post

- **One-sentence claim** — what the author actually argues, not what the headline implies.
- **Three supporting beats** — the load-bearing points.
- **What's new** — what the article adds vs prior coverage; if nothing, say so.

### Transcript (interview, podcast)

- **Speaker map** — who's who, in one line each.
- **Top points** — 5–7 bullets of the most substantive content; quote sparingly, paraphrase mostly.
- **Notable quotes** — only if a sentence is so well-said that paraphrasing would lose the value.

### Length discipline

- If the brief specifies a length, hit it.
- If not, default to the shortest faithful summary. Pad, and the reader doesn't trust you.
- If the source has thin content, return a short summary and say so. Manufactured substance is worse than honesty.

### Universal rules

- Quote sparingly, paraphrase mostly. Never invent a quote.
- Preserve disagreements; don't flatten them into false consensus.
- Attribute when attribution matters (decisions, controversial claims). Strip attribution when it doesn't (uncontroversial facts).
- Translate jargon when the audience won't share it.
